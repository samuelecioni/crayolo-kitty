#### crayolo
This is a port of [dkeg/crayolo](https://github.com/dkeg/crayolo) to [kovidgoyal/kitty](https://github.com/kovidgoyal/kitty).

All credits goes to original author.

#### Implementation
  To use a colorscheme, simply include it at the end of your kitty.conf.
	Example:
  ```
	include ~/.config/crayolo-kitty/colors/simplicity
```
